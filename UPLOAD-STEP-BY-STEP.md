# Upload Step-by-Step

## Do this slowly. No rush.

### 1. Open GitHub
Go to the `website_dev` repository.

### 2. Click Add file
Click the **Add file** button near the top right of the file list.

### 3. Click Upload files
Choose **Upload files**.

### 4. Drag the extracted files
Open the extracted Version 3 folder on your computer.

Select everything inside the folder:

- admin
- assets
- content
- index.html
- services.html
- projects.html
- service-area.html
- about.html
- contact.html
- style.css
- script.js
- netlify.toml
- README-FIRST.md
- other included files

Drag all of that into GitHub.

### 5. Wait
Let GitHub finish uploading.

### 6. Commit
At the bottom, use this message:

`Upload Version 3 website`

Click **Commit changes**.

### 7. Wait for Netlify
Netlify should automatically publish the new version in about 1–2 minutes.

### 8. Test the site
Open your Netlify temporary URL.

Check:

- Home
- Services
- Projects
- Service Area
- About
- Contact

### 9. Do not change DNS yet
Leave Wix and the live domain alone until the temporary Netlify site is perfect.
